# Security Policy

## Supported Versions

The `main` branch is currently supported with security updates.

## Reporting a Vulnerability

If you discover a security vulnerability, please do **not** open a public issue.

Instead, report it using **GitHub Security Advisories**:

- Go to the repository
- Click on **Security**
- Select **Report a vulnerability**

We will review the report and respond as soon as possible.

Thank you for helping keep Reviselyt secure.
