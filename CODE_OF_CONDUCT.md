# Code of Conduct

## Our Pledge

We are committed to providing a welcoming and respectful environment for everyone.

Participation in this project should be free from harassment or discrimination of any kind.

---

## Our Standards

Positive behavior includes:
- Respectful communication
- Constructive feedback
- Empathy toward others
- Collaborative problem-solving

Unacceptable behavior includes:
- Harassment or discrimination
- Personal attacks or insults
- Trolling or hostile language
- Sharing private information without consent

---

## Enforcement

Project maintainers may take appropriate action against behavior that violates this Code of Conduct.

This may include warnings, removal of contributions, or banning participants.

---

## Scope

This Code of Conduct applies to:
- GitHub issues and pull requests
- Discussions and community interactions
- Any official project communication

---

## Reporting

If you experience unacceptable behavior, please report it privately to the project maintainer.

All reports will be reviewed confidentially and fairly.
